# chemistry

Personal chemistry notes — Química Orgânica, Geral/Inorgânica e Físico-Química.

Built with [VitePress](https://vitepress.dev/) + [Carbon Theme](https://github.com/brenoepics/vitepress-carbon). Deployed on Vercel.

🌐 [chemistry.marlonwq.tech](https://chemistry.marlonwq.tech)

## Content

- **Química Orgânica** — Estrutura molecular, nomenclatura, hidrocarbonetos, funções, reações, isomeria
- **Química Geral e Inorgânica** — Substâncias, leis das reações, grandezas, estequiometria, gases, ligações
- **Físico-Química** — Termodinâmica, calorimetria, termoquímica, cinética, equilíbrio, coloides, soluções

## Development

```bash
npm install
npm run docs:dev
```
